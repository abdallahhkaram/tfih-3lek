# **App Name**: SafeSpot

## Core Features:

- Incident Reporting Form: Report new incidents with description and photo upload.
- Interactive Map: Pinpoint incident locations on a map.
- Incident Details: View details of individual incidents through clickable markers.
- Dynamic Heatmap Overlay: Visualize incident density with a heatmap.
- AI Incident Classification Tool: Use AI to automatically classify reported incidents based on their description and images. This helps in prioritizing and routing incidents to the appropriate authorities, using an LLM as a tool to determine when an incident merits an escaltion for human verification.

## Style Guidelines:

- Primary color: Map-centric blue (#4681C4) to invoke safety and reliability.
- Background color: Soft grey (#E9ECEF) for a clean and neutral backdrop.
- Accent color: Vibrant orange (#F29441) for the heatmap and CTAs.
- Body and headline font: 'PT Sans', a modern humanist sans-serif providing clarity for all text elements.
- Use consistent, minimalist icons for incident types and map controls.
- Responsive layout ensuring accessibility across devices.
- Subtle animations on map interactions and when submitting new reports.
